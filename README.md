## ✨ Features

- **🎯 Modern Hero Section** — Interactive landing section with animated UI elements.
- **🪄 3D Parallax Effects** — Mouse-follow tilt effect on the profile image.
- **🃏 3D Project Cards** — Smooth hover tilt animations on project cards.
- **🌌 Animated Background** — Grid lines, glowing orbs, and dynamic visual effects.
- **✨ Scroll Reveal Animations** — Sections animate into view while scrolling.
- **📊 Animated Skill Bars** — Progress bars smoothly fill on scroll.
- **📄 CV Modal** — Built-in animated resume modal with detailed information.
- **🌓 Dark / Light Theme** — Fully functional theme switcher.
- **📱 Fully Responsive** — Optimized for desktop, tablet, and mobile devices.
- **🍔 Mobile Navigation** — Responsive hamburger menu for smaller screens.
- **⚡ Zero Dependencies** — No frameworks or build tools required.

## 🎨 Customization

All editable content is stored inside:

```bash
js/data.js
```

You can easily update:

* Personal information
* Skills
* Projects
* Experience
* Social links
* Stats
* Resume content

### Change Profile Photo

Replace:

```bash
assets/profile.jpg
```

with your own image.

### Change Theme Colors

Edit:

```bash
css/variables.css
```

Example:

```css
:root {
  --primary: #3b82f6;
  --bg: #080c18;
  --card: #0f1629;
}
```
